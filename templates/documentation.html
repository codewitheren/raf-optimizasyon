Başlangıç Kılavuzu

SmartStore AI'ya hoş geldiniz! Bu kılavuz size sistemin temel özelliklerini öğretecek ve hızla başlamanızı sağlayacak.

🚀 Hızlı Başlangıç

  1. CSV Dosyanızı Hazırlayın: Ürün isimlerini içeren bir CSV dosyası oluşturun
  2. Model Seçin: Naive Bayes, Decision Tree veya Logistic Regression
  3. Analiz Edin: Dosyanızı yükleyin ve sonuçları bekleyin
  4. Optimizasyon Yapın: Raf düzeni önerilerini uygulayın

Makine Öğrenmesi Modelleri

Sistemimiz üç farklı algoritma sunar. Her birinin kendine özgü avantajları vardır:

🧠 Naive Bayes

Doğruluk: ~89%

Hız: Çok hızlı

Kullanım: Metin sınıflandırma

Avantaj: Küçük veri setlerinde etkili

🌳 Decision Tree

Doğruluk: ~87%

Hız: Hızlı

Kullanım: Kategorik veriler

Avantaj: Karar sürecini görselleştirme

📈 Logistic Regression

Doğruluk: ~91%

Hız: Orta

Kullanım: Linear ayrılabilir veriler

Avantaj: İstatistiksel güvenilirlik

API Dokümantasyonu

🔑 Authentication

API anahtarınızı header'da gönderin:

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

📤 Endpoints

Tekli Tahmin

POST /api/v1/predict 
{
  "product_name": "Coca Cola 330ml",
  "model": "logistic_regression"
}

Toplu Tahmin

POST /api/v1/predict/bulk 
{
  "products": ["Coca Cola", "Ülker Çikolata", "Sütaş Süt"],
  "model": "naive_bayes"
}

Birliktelik Analizi

POST /api/v1/association 
{
  "baskets": [
    ["Coca Cola", "Cips"],
    ["Süt", "Ekmek", "Tereyağı"]
  ],
  "min_support": 0.1,
  "min_confidence": 0.6
}
